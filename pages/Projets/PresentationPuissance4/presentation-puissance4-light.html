00 Contexte du projet

Puissance 4 est l'un de mes projets d'apprentissage fondamental : l'occasion de maîtriser les listes 2D, les fonctions, la boucle principale d'un jeu et la détection de victoire. Il a été livré en deux versions successives, puis réinventé côté web pour cette page — avec de la 3D et une timeline 4D.

Version 1 — Console (CM)

Sortie lors des cours magistraux. Pas d'interface graphique : tout se passe dans le terminal, avec une grille ASCII (. pour case vide, X et O pour les joueurs). Niveau débutant, style scolaire/BTS.

Version 2 — Graphique Tkinter

Évolution pensée UX : choix des couleurs, plateau bleu, jetons circulaires, survol souris. La même logique de jeu, habillée d'un Canvas Tkinter et d'un écran de configuration avant la partie.

Version 3 — Web 3D/4D

La présentation que tu parcours : un plateau 3D Three.js qui rejoue des parties, un visualiseur 4D (3D + temps) et une démo jouable en HTML/CSS/JS, sans framework, directement dans le navigateur.

02 Du clic au jeton : le pipeline

Saisie

En console, input() demande une colonne 1–7. Dans la version Tkinter, un <Button-1> sur le Canvas est converti en index de colonne via la position event.x.

Gravité

inserer_jeton parcourt la colonne du bas vers le haut : on pose le jeton dans la première case vide rencontrée. Simple, efficace, 100% déterministe.

Victoire

verifier_direction compte les jetons alignés dans une direction (et son opposée). Appelée sur les 4 axes : horizontal, vertical, diagonale montante et descendante.

Tour suivant

Si personne ne gagne et la grille n'est pas pleine, on alterne X ↔ O (ou joueur 1 ↔ 2). La boucle principale reprend jusqu'au verdict final.

04 Extraits de code

Les pièces maîtresses côté Python : gravité, détection de victoire, survol souris sur le plateau Tkinter. Tout est en Python standard, sans dépendance externe.

Gravité : insérer un jeton

On balaie la colonne du bas vers le haut et on dépose le jeton dans la première case libre. Pas de physique, juste une boucle for descendante.

puissance4.py 
def inserer_jeton(grille, colonne, jeton):
    indice_colonne = colonne - 1
    for ligne in range(NB_LIGNES - 1, -1, -1):
        if grille[ligne][indice_colonne] == VIDE:
            grille[ligne][indice_colonne] = jeton
            return ligne
    return -1

Détection de victoire dans 4 directions

verifier_direction compte les jetons identiques dans un sens et dans l'opposé. On appelle la fonction avec 4 couples (dLigne, dColonne) : horizontal, vertical, les deux diagonales.

puissance4.py 
def verifier_direction(grille, ligne, colonne, jeton, dL, dC):
    compteur = 1
    l, c = ligne + dL, colonne + dC
    while 0 <= l < NB_LIGNES and 0 <= c < NB_COLONNES and grille[l][c] == jeton:
        compteur += 1
        l += dL; c += dC
    l, c = ligne - dL, colonne - dC
    while 0 <= l < NB_LIGNES and 0 <= c < NB_COLONNES and grille[l][c] == jeton:
        compteur += 1
        l -= dL; c -= dC
    return compteur >= 4

def verifier_victoire(grille, ligne, colonne, jeton):
    for dL, dC in [(0, 1), (1, 0), (1, 1), (1, -1)]:
        if verifier_direction(grille, ligne, colonne, jeton, dL, dC):
            return True
    return False

Survol souris & preview du jeton (Tkinter)

Dans la version graphique, un jeton "fantôme" apparaît en haut de la colonne survolée, dans la couleur du joueur courant. On transforme event.x en index de colonne.

def x_vers_colonne(self, x):
    if x < MARGE_X or x > MARGE_X + NB_COLONNES * TAILLE_CASE:
        return None
    return int((x - MARGE_X) // TAILLE_CASE)

def survol_souris(self, event):
    if self.partie_terminee:
        return
    self.colonne_survolee = self.x_vers_colonne(event.x)

Plateau Three.js (extrait de la page)

Le plateau bleu est un BoxGeometry, chaque case est un CircleGeometry sombre, les jetons sont des CylinderGeometry. La chute est intégrée par gravité simulée et rebond décroissant.

presentation-puissance4.js 
const tokenGeo = new THREE.CylinderGeometry(R, R, 0.22, 48);
tokenGeo.rotateX(Math.PI / 2);

function makeToken(color) {
    return new THREE.Mesh(tokenGeo, new THREE.MeshPhongMaterial({
        color, emissive: color, emissiveIntensity: 0.22,
        shininess: 90, specular: 0xffffff,
    }));
}

// Gravité + rebond décroissant
token.userData.vy += GRAVITY;
token.position.y += token.userData.vy;
if (token.position.y <= yTarget) {
    if (bouncesLeft > 0) { vy = -vy * 0.35; bouncesLeft--; }
    else { token.userData.resting = true; }
}
