La Genèse du Projet

MAUICoach1 est né d'une volonté simple : centraliser le calcul de l'Indice de Masse Grasse (IMG) dans une interface mobile épurée et immédiate. En reprenant les données fondamentales (poids, taille, âge, sexe), l'application délivre un diagnostic instantané et visuel.

L'Essentiel, sans compromis.

  • Saisie Ergonomique
    Claviers numériques natifs pour la rapidité.
  • Calcul Précis
    Algorithme médicalement sourcé.
  • Validation Fluide
    Traitement des erreurs (vide, négatif) sans crash.

Architecture Infaillible

Tout repose sur la classe Profil. En isolant la logique de calcul de l'interface graphique (principe fondamental MVVM approché), l'application gagne en maintenabilité et en fiabilité.

XAML : L'Art du Pixel

Structurée autour de VerticalStackLayout et de composants natifs, l'interface s'adapte majestueusement à toutes les tailles d'écrans. Le code XAML reste déclaratif, clair et concis.

C# : Le Moteur Invisible

Derrière chaque tap, un gestionnaire d'événements performant prend le relais. La logique en arrière-plan valide, instancie le profil, et met à jour l'interface avec des feedbacks visuels instantanés.

public class Profil
{
    private int sexe;
    private double poids, taille;
    private int age;
    private double img;
    private string message;

    public Profil(int sexe, double p, double t, int a)
    {
        this.sexe = sexe;
        this.poids = p;
        this.taille = t;
        this.age = a;
        CalculIMG();
        ResultatIMG();
    }

    private void CalculIMG()
    {
        double tMetres = taille / 100.0;
        img = (1.2 * poids / (tMetres * tMetres))
            + (0.23 * age) - (10.83 * sexe) - 5.4;
    }
}

private void AfficherResultat(Profil profil)
{
    string msg = $"Votre IMG : {profil.Img:F2}%. {profil.Message}";
    lblResultat.Text = msg;
    lblResultat.IsVisible = true;
    
    if (profil.Message == "Trop maigre.") 
        imgResultat.Source = "smiley_trop_maigre";
    else if (profil.Message == "Parfait.") 
        imgResultat.Source = "smiley_parfait";
    else 
        imgResultat.Source = "smiley_surpoids";
        
    imgResultat.IsVisible = true;
}
