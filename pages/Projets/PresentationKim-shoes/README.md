# Présentation KimShoes

Ce dossier contient la page de présentation du projet **Kim-shoesPF** (KmshoesVF).

## Vue d'ensemble

La présentation met en avant :
- **Hero** : Titre, description, boutons CTA et mockup du navigateur
- **Fonctionnalités clés** : Catalogue paginé, Panier dynamique, Processus commande
- **Architecture MVC** : Entités relationnelles, sécurité, extensions Twig
- **Extraits de code** : CartService.php et CheckoutController.php

## Visualisation

Ouvrir le fichier `presentation-kim-shoes.html` directement dans un navigateur :

```
open presentation-kim-shoes.html
```

Ou démarrer un serveur local :

```bash
# Avec Python
python3 -m http.server 8080

# Avec PHP (depuis le dossier Presentation)
php -S localhost:8080
```

Puis accéder à : http://localhost:8080
